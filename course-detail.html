Nội dung bài học

Trong bài học này, học sinh sẽ được ôn lại cấu trúc cơ bản của một trang HTML. Đây là phần kiến thức nền tảng trước khi học CSS và Bootstrap 5.

1. Cấu trúc cơ bản của trang HTML

Một trang HTML cơ bản thường gồm các phần chính như sau:

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>Trang web đầu tiên</title>
</head>
<body>
    <h1>Xin chào HTML</h1>
    <p>Đây là đoạn văn bản đầu tiên.</p>
</body>
</html>
2. Một số thẻ HTML thường dùng
Thẻ HTML Công dụng
<h1> - <h6> Tạo tiêu đề cho nội dung.
<p> Tạo đoạn văn bản.
<img> Hiển thị hình ảnh.
<a> Tạo liên kết đến trang khác.
<div> Dùng để chia bố cục trang web.
3. Ghi nhớ
HTML dùng để xây dựng cấu trúc nội dung của trang web. CSS dùng để trang trí giao diện, còn Bootstrap 5 giúp thiết kế giao diện nhanh và responsive hơn.
Giới thiệu bài học

Trong bài học này, học sinh sẽ ôn lại cấu trúc cơ bản của một trang HTML, cách sử dụng thẻ tiêu đề, đoạn văn, hình ảnh, liên kết và bố cục nội dung.

Mục tiêu sau bài học
  • Biết tạo cấu trúc cơ bản của file HTML.
  • Biết sử dụng các thẻ HTML thông dụng.
  • Biết liên kết CSS vào trang HTML.
  • Chuẩn bị kiến thức để học Bootstrap 5.
